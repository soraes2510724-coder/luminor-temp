Heap Sort (힙 정렬)

힙 정렬은 최대 힙(또는 최소 힙) 구조를 이용해 가장 큰(또는 작은) 값을 반복적으로 꺼내며 정렬하는 알고리즘입니다.

🧠 작동 원리

1️⃣ 배열을 힙 구조로 변환합니다.
2️⃣ 루트(최댓값)를 끝으로 보내고 힙 크기를 줄입니다.
3️⃣ 다시 힙을 재구성하고 반복합니다.

⚙️ 시간 복잡도

모든 경우 O(n log n)

✅ 장단점

  • 추가 메모리 거의 필요 없음
  • 안정 정렬이 아님
  • 성능이 일정하게 우수함
