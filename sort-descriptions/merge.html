Merge Sort (병합 정렬)

병합 정렬은 배열을 절반으로 분할하고 각각을 정렬한 후, 병합하여 전체를 정렬하는 분할 정복(Divide and Conquer) 알고리즘입니다.

🧠 작동 원리

1️⃣ 배열을 절반으로 나눕니다.
2️⃣ 각 부분 배열을 재귀적으로 정렬합니다.
3️⃣ 두 부분을 병합하며 하나의 정렬된 배열을 만듭니다.

⚙️ 시간 복잡도

모든 경우 O(n log n)

✅ 장단점

  • 안정 정렬 (stable sort)
  • 예측 가능한 시간 복잡도
  • 추가 메모리 사용량이 많음
